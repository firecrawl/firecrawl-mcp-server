## ðŸ› Troubleshooting

### Wake Word Not Working

**Problem**: Wake word not triggering

**Solutions**:
1. Use Chrome, Edge, or Safari
2. Allow microphone permissions
3. Speak clearly: "Hey Monster AI"
4. Check green indicator is on
5. Ensure no other app using mic

### Code Not Highlighting

**Problem**: Code blocks not colored

**Solutions**:
1. Check toggle is enabled
2. Ensure internet connection (CDN)
3. Refresh page
4. Check browser console for errors

### Export Not Downloading

**Problem**: Export button doesn't download

**Solutions**:
1. Check popup blocker
2. Allow downloads in browser
3. Check browser download settings
4. Try different format

### Typing Effect Slow

**Problem**: Responses appear too slowly

**Solutions**:
1. Disable typing effect toggle
2. Or edit typing speed (see Advanced Configuration)
