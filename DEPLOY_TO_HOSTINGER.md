## ğŸ”§ Troubleshooting

### **404 Error - Page Not Found**

**Cause**: File not named correctly or in wrong folder

**Fix:**
1. Check file is named exactly `index.html` (lowercase!)
2. Check it's in the subdomain folder
3. Clear browser cache and try again

### **Intro Plays Every Time**

**Cause**: Using Incognito/Private browsing

**Fix:**
- Normal browsing mode remembers you've seen it
- Or click "Skip Intro" button

### **Microphone Not Working**

**Cause**: Browser permissions or no HTTPS

**Fix:**
1. Enable SSL certificate (HTTPS)
2. Allow microphone when browser prompts
3. Use Chrome, Edge, or Safari (best support)

### **API Key Not Saving**

**Cause**: Browser blocking localStorage

**Fix:**
1. Ensure HTTPS is enabled
2. Check browser allows cookies/storage
3. Try different browser
4. Re-enter and click Save
