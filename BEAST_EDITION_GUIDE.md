## ğŸ› Troubleshooting

### **Server won't start:**
```bash
# Check if port 5001 is in use
netstat -ano | findstr :5001  # Windows
lsof -i :5001  # Mac/Linux

# Kill the process or change PORT in .env.ultimate
```

### **Frontend can't connect:**
- Verify server is running on `http://localhost:5001`
- Check browser console for errors
- Make sure CORS is enabled (already configured)

### **Service shows as inactive:**
- Verify API key is correct in `.env.ultimate`
- Check API key has credits/is active
- View server logs for error details

### **Microphone not working:**
- Browser must be HTTPS (or localhost)
- Allow microphone permissions
- Chrome/Edge work best

### **LiveKit connection fails:**
- Verify LiveKit credentials are correct
- Check LiveKit cloud status
- Ensure network allows WebRTC
