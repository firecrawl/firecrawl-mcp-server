# @firecrawl/cli

Firecrawl CLI for web scraping, crawling, search, extraction, and agent workflows (no MCP tool-description overhead).

## Quickstart

```bash
export FIRECRAWL_API_KEY=fc-YOUR_API_KEY
npx -y -p @firecrawl/cli firecrawl scrape https://example.com --json '{"formats":["markdown"]}'
```
