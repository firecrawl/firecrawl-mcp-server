# firecrawl-mcp

Model Context Protocol (MCP) server for Firecrawl web scraping, crawling, search, and extraction.

## Quickstart (stdio)

```bash
export FIRECRAWL_API_KEY=fc-YOUR_API_KEY
npx -y firecrawl-mcp
```
